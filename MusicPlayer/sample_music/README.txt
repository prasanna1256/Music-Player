Sample Music Directory
======================

This directory is intended to store audio files for testing the music player.

Supported file formats:
- WAV (.wav)
- MP3 (.mp3)
- AIFF (.aiff, .aif)
- FLAC (.flac)
- OGG (.ogg)

How to use:
1. Place your audio files in this directory
2. Launch the music player application
3. Add songs to your playlist using the "Add Song" button or import the entire directory

Note: No sample audio files are included with this project due to copyright concerns.
You will need to provide your own audio files for testing purposes.
